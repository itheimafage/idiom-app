# 花生成语800词 - 产品需求文档

## 产品概述

一款专注于成语学习的微信小程序，收录800个精选成语，提供释义、出处、例句等完整学习内容。支持搜索、收藏、学习进度跟踪，以中国传统水墨风格呈现，温暖的花生色系营造沉浸式学习体验。

## 核心功能

### 1. 成语列表浏览
- 800个成语分页展示，每页20条
- 支持按拼音首字母（A-Z）分类筛选
- 支持按难度等级筛选（入门/进阶/高级）
- 列表展示成语名称、拼音、简短释义
- 下拉刷新，上拉加载更多

### 2. 搜索功能
- 支持成语名称搜索（模糊匹配）
- 支持拼音搜索
- 支持关键词搜索释义内容
- 搜索结果高亮显示
- 搜索历史记录

### 3. 成语详情页
- 成语大字展示（带拼音注音）
- 详细释义说明
- 出处典故
- 例句展示（2-3条）
- 近义词/反义词
- 难度等级标签
- 收藏按钮
- 已学标记按钮

### 4. 收藏功能
- 收藏/取消收藏成语
- 收藏列表独立页面
- 支持从收藏中移除
- 收藏数据本地持久化存储

### 5. 学习进度
- 标记成语为"已学"
- 显示总学习进度百分比
- 已学成语列表
- 学习进度本地持久化

### 6. 每日推荐
- 首页展示"每日成语"卡片
- 随机推荐一个成语供学习

## 页面结构

| 页面路径 | 页面名称 | 功能描述 |
|---------|---------|---------|
| pages/index/index | 首页 | 每日成语推荐、成语分类入口、搜索入口 |
| pages/list/list | 成语列表 | 分页成语列表、分类筛选、搜索 |
| pages/detail/detail | 成语详情 | 成语完整信息展示、收藏/已学操作 |
| pages/favorites/favorites | 我的收藏 | 已收藏成语列表 |
| pages/learned/learned | 已学成语 | 已学成语列表、学习进度 |

  "abbr": "yxyy",

## API 接口

| 方法 | 路径 | 描述 |
|-----|------|------|
| POST | /api/idioms/list | 分页获取成语列表 |
| POST | /api/idioms/search | 搜索成语 |
| POST | /api/idioms/detail | 获取成语详情 |
| POST | /api/idioms/random | 获取随机每日成语 |
| POST | /api/idioms/categories | 获取所有分类 |

## 设计风格

- **色彩体系**：花生色系 — 米白底色(#FFF8F0)、暖棕主色(#8B6914)、花生壳色(#C49A3C)、深褐文字(#4A3728)
- **视觉风格**：中国传统水墨画风格，淡雅留白，毛笔书法字体标题
- **排版**：竖排传统韵味（详情页），列表采用清晰卡片式布局
- **动画**：柔和的翻页过渡效果，水彩晕染加载效果
